TWSSnippet

data class TWSSnippet(val id: String, val target: String, val headers: Map<String, String> = emptyMap(), val dynamicResources: List<TWSAttachment> = emptyList(), val props: Map<String, Any> = emptyMap(), val engine: TWSEngine = TWSEngine.NONE, val loadIteration: Int = 0) : Parcelable

Represents a web snippet that can be rendered in a WebView.

Parameters

id

A unique identifier for the snippet.

target

The URL of the snippets content.

headers

custom HTTP headers to include with the request.

dynamicResources

A list of resources (CSS/JS) to inject.

props

Custom properties for the snippet. If the engine is set to TWSEngine.MUSTACHE, these properties are also used for Mustache template processing.

engine

Specifies how the snippet content is processed, either rendered normally or processed as Mustache.

loadIteration

A counter for manual reloads of the snippet, useful whenever snippets HTML is changed.
